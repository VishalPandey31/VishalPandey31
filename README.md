### Method 2: Pin Repositories & Set "Website" Link (Sabse Aasan Tarika 📌)
Aapke screenshot mein **"Customize your pins"** likha hua hai. Wahan se aap projects ko directly profile par dikha sakte ho.
**Kaise karein:**
1. Aapne Screenshot mein dekho, right side mein **"Customize your pins"** likha hai, uspar click karo.
2. Apne best projects ko tick karke Save kardo (Jaise *GeoNest*, *ElectroMart*, etc.). Ye aapki profile par front mein dikhne lagenge.
3. Har repository ke andar jao ➔ Right side mein **About** ke aage chhota sa **⚙️ Gear icon** hoga uspar click karo ➔ **Website** wale box mein apna link (jaise `https://GeoNest.surge.sh`) paste karke save kardo. 

**Result:** Jab koi profile par aayega, toh usko Pins mein seedha click karne wala link mil jayega!

Agar aap chahte ho main locally aapke `VishalPandey31` naam ka folder banakar README likh dun aur use push karun, toh aap bas wahan GitHub par empty repository banakar mujhe uska URL de dena!
